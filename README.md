# Calculator App

A simple and responsive Calculator web-app built using JavaScript.  
This calculator perform basic arithmetic operations along with scientific functionalities like trigonometry, modulo, decimal calculation,unary operation, bracket handling, and π (pi) support.

---

## Features

### Basic Operation
- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Unary operation

### Scientific Feature
- Trigonometric Functions
  - `sin()`
  - `cos()`
  - `tan()`
- Modulo Operation (`%`)
- Decimal Calculations (`.`)
- Bracket Support (`()`)
- Pi Value (`π`)
- Clear Screen (`C`)
- Evaluate Expression (`=`)

---

## 🛠️ Technologies Used

- HTML

- JavaScript
